# SiE (Self-improvement Ecosystem)

Современная экосистема инструментов для саморазвития, построенная на принципах модульности и глубокой геймификации.

## 🛠 Текущие модули
- **Breathing Practices**: Профессиональный тренажер дыхания (Метод Вима Хофа) с динамической аудио-подстройкой и иммерсивным дизайном.
- **Habit Tracker**: Гибкий инструмент контроля привычек с адаптивным интерфейсом и системой вознаграждений.
- **Hall of Fame**: Централизованный профиль оперативника с отслеживанием XP, уровней и достижений.

## 🚀 Технологии
- Flutter & Riverpod
- Supabase (Backend & Auth)
- Docker (Local Development)

## 📦 Структура
- `central_hub`: Основное приложение и профиль.
- `sie_core`: Ядро системы и общие компоненты.

## 🔧 Установка
1. `git clone [url]`
2. `supabase start`
3. `flutter pub get`
4. `flutter run`
