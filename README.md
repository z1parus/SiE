**SiE** — это высокотехнологичная экосистема для саморазвития, выполненная в эстетике «киберпанк-оперативного штаба». Приложение превращает дисциплину и личностный рост в увлекательный процесс управления сложной системой, где каждая привычка — это протокол, а каждая цель — тактическая миссия.

---

## 🚀 Основные возможности (Реализовано)

### 1. Operations Control (Главный штаб)
*   **Интерактивный дашборд:** Визуализация текущего уровня XP, статуса оператора и активных департаментов.
*   **XP System:** Геймифицированная прогрессия. Выполняйте задачи, чтобы повышать уровень от "Recruit" до "Commander".
*   **Branch Navigation:** Быстрый доступ к ключевым модулям через систему «Департаментов».

### 2. Habit Matrix (Управление привычками)
*   **Система протоколов:** Создание и отслеживание ежедневных привычек.
*   **Habit Routines:** Гибкая настройка расписания (по дням недели, интервалам).
*   **Journaling:** Возможность оставлять заметки к каждому логу привычки для ретроспективы.
*   **Архивация и закрепление:** Управление фокусом через приоритизацию важных протоколов.

### 3. Focus Protocol (Таймер продуктивности)
*   **Pomodoro-модификация:** Настраиваемые циклы работы и отдыха в стиле «загрузки данных».
*   **Deep Work:** Иммерсивный интерфейс, минимизирующий отвлечения.

### 4. Breathing Practices (Ментальное восстановление)
*   **Liquid Glass Visualizer:** Уникальный визуальный движок для медитативных дыхательных упражнений.
*   **Аудио-сопровождение:** Синхронизированные звуковые эффекты для вдоха, выдоха и задержки дыхания.

### 5. Interface Hub (Кастомизация)
*   **Shop System:** Магазин визуальных улучшений за Design Points (DP).
*   **Personalization:** Кастомизация рамок аватаров, фонов профиля и стилей отображения статистики.
*   **Liquid Glass Engine:** Динамическое изменение визуальной темы приложения (Cosmic Mode).

### 6. Social & Hall of Fame
*   **Daily Leaderboard:** Глобальный рейтинг «Авангард Суток», обновляемый в реальном времени.
*   **User Search:** Поиск других оперативников в системе.
*   **Operative Dossier:** Публичные профили с отображением достижений и экипировки.

## 🛠 Технический Стек

*   **Frontend:** Flutter (Dart) — кроссплатформенная разработка.
*   **State Management:** Riverpod — современное реактивное управление состоянием.
*   **UI Engine:** Liquid Glass Widgets — кастомный движок для реализации эффектов стекла и жидкостей.
*   **Backend:** Supabase (PostgreSQL, GoTrue, PostgREST).
    *   **Edge Functions:** TypeScript (Deno) для серверной логики (например, расчет ежедневных победителей).
    *   **Realtime:** Обновление лидербордов и статусов без перезагрузки.
*   **Database:** Сложная схема с миграциями, покрывающая логику достижений, магазина, логов и кастомизации.

## 📂 Структура проекта

*   `apps/central_hub`: Основное клиентское приложение.
*   `packages/sie_core`: Общая бизнес-логика, модели данных, сервисы (Supabase, Sync, Audio) и UI-компоненты.
*   `supabase/migrations`: История развития структуры базы данных и политик безопасности (RLS).
*   `planningSources/`: Документация по развитию проекта и проектированию новых модулей.
