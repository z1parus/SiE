# SiE (Self-improvement Ecosystem)

**SiE** — это современная экосистема микро-приложений, превращающая процесс саморазвития в управление глобальной корпорацией. Забудьте о перегруженных функциях; скачивайте только те инструменты, которые нужны вам сейчас.

## 🚀 Основные особенности
- **Микро-сервисный подход**: Каждое направление саморазвития (дыхание, привычки, дневник) — это отдельный эффективный инструмент.
- **Единый профиль**: Ваш прогресс и XP синхронизируются между всеми приложениями экосистемы.
- **Геймификация**: Прокачивайте свой статус оперативника, открывайте новые уровни и получайте достижения за реальные действия.
- **Доступность**: Все инструменты доступны по единой демократичной подписке.

## 🛠 Технологии
- Flutter & Dart
- Supabase (Database & Auth)
- Riverpod (State Management)
- Docker

## 📦 Структура проекта
Проект организован по принципу монорепозитория:
- `central_hub`: Центральное приложение для управления аккаунтом.
- `sie_core`: Ядро системы с общими компонентами и логикой.

## ⚙️ Локальное развертывание
1. Установите [Flutter](https://flutter.dev) и [Docker](https://www.docker.com).
2. Склонируйте репозиторий.
3. Запустите локальный бэкенд: `supabase start`.
4. Запустите приложение: `cd apps/central_hub && flutter run`.
