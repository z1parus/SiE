# Архитектурный дизайн: Модульные узоры фона профиля

## 1. Обзор
Цель — предоставить пользователям возможность приобретать и использовать визуальные узоры (паттерны), которые накладываются поверх фона заголовка профиля. Узоры являются модульными: их можно комбинировать с любым базовым цветом или градиентом фона.

### Принципы дизайна
*   **Низкий контраст:** Узоры используют акцентный цвет фона с высокой прозрачностью (10-15%).
*   **Многослойная отрисовка:** Фон (снизу) -> Рендерер узора (посередине) -> Контент профиля (сверху).
*   **Режимы наложения:** Использование `BlendMode.screen` или `BlendMode.overlay` для создания эффекта "премиального" цифрового интерфейса.
*   **Тонкая анимация:** Возможность добавления легкого движения или пульсации для таких узоров, как "Нейронные нити".

## 2. Модель данных и База данных

### Схема базы данных (Supabase)
Создание новой таблицы `public.profile_patterns` и обновление таблицы профилей.

```sql
-- Каталог узоров
create table public.profile_patterns (
  id           uuid              primary key default gen_random_uuid(),
  slug         text              not null unique,
  name         text              not null,
  rarity       public.cosmetic_rarity not null default 'common',
  price_dp     integer           not null default 0,
  style_config jsonb             not null default '{}' -- { "pattern_type": "neural" | "low_poly" | ... }
);

-- Обновление профилей
alter table public.profiles
  add column equipped_pattern_id uuid references public.profile_patterns(id);

-- Обновление ограничений инвентаря
alter table public.user_inventory 
  drop constraint if exists user_inventory_asset_type_check;
alter table public.user_inventory 
  add constraint user_inventory_asset_type_check 
  check (asset_type in ('avatar_frame', 'profile_background', 'stat_style', 'profile_pattern'));
```

### Модели Flutter (`sie_core`)
*   **`Profile` / `PublicProfile`:** Добавление поля `String? equippedPatternId`.
*   **`AssetType`:** Добавление типа `profilePattern`.
*   **`CosmeticAsset`:** Обновление для обработки типа `profile_pattern`.
*   **`EquippedAssets`:** Обновление метода `resolve` для включения ассета узора.

## 3. Интерфейс и Отрисовка

### Рендереры узоров
Узоры реализуются через `CustomPainter` для максимальной производительности и гибкости.

1.  **Нейронные нити:** Переплетающиеся линии с узлами и редкими световыми импульсами.
2.  **Полигональная сетка (Low-Poly):** Сетка из треугольников с тонкими градиентами на вершинах.
3.  **Изометрическая сетка:** 3D-кубы в изометрической проекции.
4.  **Точечный паттерн (Dot Matrix):** Строгая сетка из мелких точек.

### Стек реализации (`_HeaderGlassCard`)
Карточка заголовка использует `Stack` для послойного размещения элементов:
1.  **Декорация контейнера:** Базовый фон (сплошной цвет или градиент).
2.  **`ProfilePatternRenderer`:** Наложение через `Positioned.fill` с использованием `BlendMode.screen` и низкой прозрачности.
3.  **`Padding` / `Column`:** Основная информация профиля (аватар, имя, статистика).

## 4. Пользовательский опыт (UX)

### Экран кастомизации
*   **Новая вкладка:** В список вкладок (Рамки, Фоны, Стили) добавлена вкладка "УЗОРЫ".
*   **Живой предпросмотр:** Выбор узора мгновенно обновляет заголовок в превью, позволяя пользователю увидеть сочетание с текущим цветом фона.

### Магазин (Interface Hub)
*   **Новая категория:** Узоры представлены как отдельные товары с ценой, зависящей от редкости.
*   **Цвета редкости:** 
    *   Обычный (Common): Точечный паттерн.
    *   Редкий (Rare): Нейронные нити.
    *   Эпический (Epic): Полигональная сетка.
    *   Легендарный (Legendary): Изометрическая сетка.
