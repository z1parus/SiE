# Экран Деталей Миссии (Mission Detail Screen) — Classic Edition

## 1. Концепция
Этот экран является основным рабочим пространством для управления конкретной Глобальной Целью. Здесь «стратегия» превращается в «тактику». Дизайн обеспечивает работу с многоуровневой структурой через два основных режима: Список и Визуальная Карта.

---

## 2. Структура Экрана

### А. Шапка Миссии (Mission Header)
*   **Элементы:**
    *   **Название цели:** Крупный заголовок.
    *   **Общий прогресс-бар:** Тонкая линия с процентом выполнения.
    *   **Временная шкала:** Дни до дедлайна.
    *   **View Switcher:** Кнопка переключения между Список (List) / Карта (Map).

### Б. Управление Под-целями (Operations & Milestones)
*   **Формат:** Иерархический список.
*   **Логика:**
    *   **Под-цель (Sub-goal):** Группа задач.
    *   **Контрольная точка (Checkpoint):** Ключевое событие.
    *   **Взаимодействие:** Развертывание/свертывание веток.

### В. Режим Визуальной Карты (Tactical Map Mode)
*   **Что это:** Полноэкранный интерактивный режим «Ментальной карты» (Mind Map).
*   **Визуализация:**
    *   **Центральный узел:** Глобальная Цель.
    *   **Ветви:** Под-цели, контрольные точки и связанные привычки.
    *   **Стиль:** Строгие геометрические фигуры (Classic Style) с четкими линиями связи.
*   **Интерактивность (Редактирование прямо на карте):**
    *   **Direct Interaction:** Нажатие на любой узел (ноду) открывает мини-панель редактирования (название, дедлайн, статус) прямо в режиме карты.
    *   **Stucture Modification:** Возможность добавлять новые элементы через кнопки «+» на связях.
    *   **Drag-and-Drop:** Перетаскивание узлов для изменения связей (например, перенос задачи в другую под-цель).
    *   **Navigation:** Плавное масштабирование (Zoom) и перемещение (Pan) по холсту.

### Г. Список Задач (Actionable Tasks)
*   **Элементы:** Чекбоксы, даты, веса сложности.

### Д. Панель Синергии Привычек (Habit Synergy Panel)
*   **Что это:** Связь миссии с ежедневной рутиной. Отображается в обоих режимах (в Карте — как отдельные узлы).

---

## 3. Управление

1.  **View Switcher:**
    *   *List View:* Классический тактический список.
    *   *Map View:* Стратегическая визуальная карта с возможностью прямого редактирования.
2.  **Quick Entry:** Поле быстрого добавления внизу экрана.

---

## 4. Визуальные решения (Classic Themes)

### Classic Dark
*   **Карта:** Темный холст, неоновые линии связи, контрастные блоки узлов.
*   **Список:** Строгая сетка, акцентные цвета для активных элементов.

### Classic Light
*   **Карта:** Светлый чертежный фон (Grid), темные линии, пастельные заливки узлов.
*   **Список:** Чистый интерфейс с мягким разделением слоев.
