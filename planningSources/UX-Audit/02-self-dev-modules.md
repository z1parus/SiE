## 2.1 — Breathing Exercise (дыхание Wim Hof)

**Сейчас:** центральный орб-шейдер, 7-фазный автомат, настройки в bottom-sheet,
пресеты в SharedPreferences, частичный XP при выходе ≥30с.

| # | Предложение | Приоритет |
|---|---|---|
| 1 | **Подтверждение на «RELEASE»** в фазе задержки — сейчас мгновенный выход из челленджа, легко промахнуться. | 🟠 P1 |
| 2 | **Защита от случайного back-swipe** во время активной сессии (`PopScope` + диалог), как в meditation_session. | 🟠 P1 |
| 3 | Визуальный прогресс циклов: сейчас только текст «CYCLE 5/30» — добавить прогресс-бар/кольцо. | 🟠 P1 |
| 4 | Сделать кнопку Restart заметнее (сейчас серый текст 55% alpha — теряется). | 🟡 P2 |
| 5 | Reduce-motion: пульс орба, heartbeat, цвето-переходы — уважать системную настройку (Этап 0). | 🟠 P1 |
| 6 | Быстрый доступ к пресетам (не прятать save/load глубоко в скролле sheet). | 🟡 P2 |
| 7 | UI-подсказка о старте heartbeat на 30с задержки (сейчас происходит «молча»). | 🟡 P2 |
| 8 | Аудио-превью при изменении громкости/треков в настройках. | 🔵 P3 |
| 9 | Настраиваемое время подготовки (сейчас хардкод 5с countdown). | 🔵 P3 |

## 2.2 — Focus Protocol (помодоро)

**Сейчас:** кольцо-таймер, фазы idle/focus/break, настройки залочены во время
сессии, result-overlay, обработка foreground/background.

| # | Предложение | Приоритет |
|---|---|---|
| 1 | Залоченные настройки во время сессии видны, но серые (opacity 0.35) + текст «LOCKED» — запутывает. Лучше **скрывать** недоступные контролы или явный «замок»-бейдж. | 🟠 P1 |
| 2 | В фазе break показывается «+0 XP» — заменить на «ОТДЫХ» / «перерыв не даёт XP», иначе выглядит как баг. | 🟠 P1 |
| 3 | Кнопка «PAUSE» не намекает на возможность resume — добавить состояние/иконку «продолжить». | 🟡 P2 |
| 4 | Result-overlay закрывается только по «START BREAK» — добавить «✕»/назад. | 🟡 P2 |
| 5 | Session-бейдж скрыт при 0 сессий — показывать «0/N» новичкам, чтобы было понятно, что сессии считаются. | 🟡 P2 |
| 6 | Reduce-motion для пульса кольца и spec-бликов. | 🟠 P1 |
| 7 | История сессий (счётчик есть — добавить просмотр трендов). | 🔵 P3 |
| 8 | Хаптика/звук на start/pause/complete. | 🔵 P3 |
