# Этап 3 — Привычки и медитация

Экраны: Habit Tracker, Habit Detail, Meditation Hub, Meditation Preflight,
Meditation Session, Meditation Preset Builder. Самые сложные потоки с риском
потери данных и наибольшим влиянием на удержание.

---

## 3.1 — Habit Tracker

**Сейчас:** рутины (утро/вечер) + список привычек, 3 режима (today/week/allTime),
swipe-карточки, pull-to-refresh, ручной онбординг-оверлей.

| # | Предложение | Приоритет |
|---|---|---|
| 1 | **Swipe-действия недискаверабельны** — добавить визуальный hint (peek-анимация при первом входе) или явные кнопки. | 🟠 P1 |
| 2 | Иерархия «рутины vs отдельные привычки» визуально смешана — разделить секциями/заголовками. | 🟠 P1 |
| 3 | Режим (today/week/allTime) сбрасывается, не персистится — сохранять выбор. | 🟡 P2 |
| 4 | Tooltip/Semantics на иконки top-bar (archive/info). | 🟡 P2 |
| 5 | Скелетон списка вместо спиннера. | 🟡 P2 |
| 6 | Доступный путь «пропустить» в онбординг-оверлее. | 🟡 P2 |
| 7 | Inline-архивация на карточке (сейчас только через отдельную навигацию). | 🔵 P3 |

## 3.2 — Habit Detail

**Сейчас:** 7-дневная сетка, статы (streak/today/total), быстрые действия,
журнал рефлексий. Архив без подтверждения, заметка только после выполнения.

| # | Предложение | Приоритет |
|---|---|---|
| 1 | **Архивация без подтверждения** (3128) — добавить `confirmDestructive` (Этап 0). | 🔴 P0 |
| 2 | 7-дневная сетка выглядит кликабельной, но read-only — либо дать ретро-логирование по тапу на день, либо убрать ложный аффорданс. | 🟠 P1 |
| 3 | Заметку/рефлексию разрешить **до** выполнения (планирование намерения), не только после. | 🟠 P1 |
| 4 | Disabled-состояние кнопки «ЗАМЕТКА» слишком слабое (только fade) — явный disabled-стиль + tooltip почему недоступно. | 🟡 P2 |
| 5 | Settings-иконка 18px серая, низкий контраст → увеличить зону + контраст. | 🟡 P2 |
| 6 | Inline быстрый edit имени/цвета без полной модалки. | 🔵 P3 |
| 7 | В журнале показывать и выполненные-без-заметки дни (сейчас невидимы). | 🔵 P3 |

## 3.3 — Meditation Hub

**Сейчас:** мини-статы (zen streak / минуты / Clarity XP), quick-start стрип,
фильтры, список пресетов, long-press → действия. Сортировка по часу дня.

| # | Предложение | Приоритет |
|---|---|---|
| 1 | **Удаление пресета без подтверждения/undo** (574) — Этап 0. | 🔴 P0 |
| 2 | Empty-state «Пресеты не найдены» показывается и когда пресеты есть, но отфильтрованы — различать «нет данных» vs «нет по фильтру» + сброс фильтра. | 🟠 P1 |
| 3 | Невидимая сортировка по часу дня сбивает с толку (порядок «прыгает») — показать ярлык «Рекомендовано сейчас» над приоритетной группой. | 🟠 P1 |
| 4 | Long-press недискаверабелен — добавить «⋮»-кнопку на карточке как альтернативу. | 🟠 P1 |
| 5 | Fire-иконка `0xFFFF6B35` → токен темы. | 🟡 P2 |
| 6 | Избранное/pin пресетов (помимо quick-start). | 🟡 P2 |
| 7 | Персистентность фильтра между сессиями. | 🔵 P3 |
| 8 | Описание в quick-start карточках (сейчас только длительность). | 🔵 P3 |

## 3.4 — Meditation Preflight

**Сейчас:** настройка перед сессией (структура, длительности, аудио-слайдеры,
аффирмации). Back теряет изменения без предупреждения.

| # | Предложение | Приоритет |
|---|---|---|
| 1 | Back при изменённых настройках — без предупреждения теряет кастомизацию. Добавить «Сохранить как пресет?»/подтверждение. | 🟠 P1 |
| 2 | Объяснить паттерны дыхания: «4-7-8», «Box 4×4» — что значат цифры (секунды), tooltip/подпись «вдох-задержка-выдох-задержка». | 🟠 P1 |
| 3 | Кнопка «Сброс к исходному пресету». | 🟡 P2 |
| 4 | Визуальный индикатор дефолтов слайдеров (0.7/0.5/0.6) — легко случайно сдвинуть. | 🟡 P2 |
| 5 | Тонкая настройка интервала аффирмаций (сейчас шаг 10с). | 🔵 P3 |
| 6 | Индикатор текущего выбранного пака аффирмаций. | 🔵 P3 |

## 3.5 — Meditation Session (иммерсивный экран)

**Сейчас:** орб (teal-градиенты хардкод), прогресс-дуга по углам, аффирмации,
плавающий микшер (auto-hide), dark-screen режим, completion-sheet с настроением.

| # | Предложение | Приоритет |
|---|---|---|
| 1 | **Dark-screen режим скрывает всё, включая таймер** — оставить минимальный видимый таймер/способ выхода, иначе пользователь «теряется». | 🟠 P1 |
| 2 | Tap-по-орбу для микшера недискаверабелен — добавить тонкую подсказку при старте (fade-хинт «коснитесь для микшера»). | 🟠 P1 |
| 3 | Нет индикатора паузы на экране — показать состояние «PAUSED» поверх орба. | 🟠 P1 |
| 4 | Порог частичного зачёта 60с — пользователь может «молча» потерять 59с. Показывать заранее «прогресс засчитается после 1 мин». | 🟡 P2 |
| 5 | Аффирмация снизу перекрывается микшером при его появлении — развести по вертикали. | 🟡 P2 |
| 6 | reflectionPause-оверлей без таймера («Подготовка к рефлексии…») — добавить отсчёт. | 🟡 P2 |
| 7 | Прогресс-дуга в углах малозаметна — добавить текстовый % / MM:SS прогресса. | 🟡 P2 |
| 8 | Эмодзи-селектор настроения без текстовых подписей — добавить label (a11y). | 🟡 P2 |
| 9 | Reduce-motion: пульс/opacity орба. | 🟠 P1 |
| 10 | Орб-цвета teal вынести в токен (сейчас не кастомизируются темой). | 🔵 P3 |

## 3.6 — Meditation Preset Builder (мастер из 4 шагов)

**Сейчас:** PageView без свайпа, нижняя навигация Back/Next/Save, валидация
имени по snackbar, back теряет изменения.

| # | Предложение | Приоритет |
|---|---|---|
| 1 | Back на любом шаге закрывает мастер без предупреждения — потеря несохранённого. Добавить confirm. | 🟠 P1 |
| 2 | Валидация имени — snackbar после Next; лучше inline-ошибка под полем on blur. | 🟠 P1 |
| 3 | Нет «тест-запуска» пресета перед сохранением (в отличие от Preflight). Добавить «Попробовать». | 🟡 P2 |
| 4 | Объяснение паттернов дыхания (как в 3.4). | 🟡 P2 |
| 5 | Различать обязательные/опциональные поля визуально. | 🟡 P2 |
| 6 | Превью пака аффирмаций до выбора (сейчас только после). | 🔵 P3 |
| 7 | Нет success-тоста после сохранения — добавить. | 🔵 P3 |
